# 💳 Credit Risk Decision System

An end-to-end machine learning system that predicts loan default risk and supports business decision-making using explainable AI.

---

## 🚀 Features
- Predicts Probability of Default (PD) using XGBoost
- Feature engineering using credit bureau data
- Business decision engine (Approve / Reject)
- SHAP-based explainability for transparent predictions
- Interactive Streamlit dashboard

---

## 📊 Model Performance
- ROC-AUC Score: ~0.75

---

## 🧠 Tech Stack
- Python, Pandas, NumPy
- Scikit-learn, XGBoost
- SHAP (Explainable AI)
- Streamlit (Dashboard)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
